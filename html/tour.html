le jeu

les régle de base

durant une partie yugioh chaque joueur commence avec 5 carte en main,le jeu se joue tour par tour,ces cartes viennent du "main deck".durant chaque tour le joueur peut jouer ces carte et attaqué, la partie est terminez quand un joueur a 0 point de vie ou qu'il n'a plus de carte dans son deck et qu'il dois pioché.

  2. la stamby phase

    4. la main phases 1 et sont identique c est les phases ou le joueur du tour peuvent jouer des cartes.par exemple: invoqué normalement un monstre,invoqué spécialement 1 ou plussieurs monstre,activé des magies ou pieges.

    5. la battle phase est la phase de combat c est a ce moment ou le joueur du tour peut faire attaqué ces personnage sur les personnage adverse,si il n as pas de personnage il peut attaqué directement l'adversaire.
